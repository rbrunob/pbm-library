# PBM Library

## About

A React component library to implement the PBM (Pharmacy Benefit Manager) model.

## How use

```
npm install @agradadigital/pbm |
yarn add @agradadigital/pbm | 
pnpm add @agradadigital/pbm
```

## Technologies Used

- React;
- Vite;
- Typescript;
- Jest;
- Storybook;
- Tailwind;
- pnpm;
